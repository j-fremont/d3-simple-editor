
# d3-simple-editor

Create and load dependencies.

```bash
$ npm install -g create-react-app
...
$ create-react-app d3-simple-editor
...
$ cd d3-simple-editor
[d3-simple-editor]$ npm install --save react react-dom
[d3-simple-editor]$ npm install --save bootstrap reactstrap
[d3-simple-editor]$ npm install --save d3
[d3-simple-editor]$ npm install --save mongoose express cors axios
[d3-simple-editor]$ npm install --save @fortawesome/fontawesome-svg-core
[d3-simple-editor]$ npm install --save @fortawesome/free-solid-svg-icons
[d3-simple-editor]$ npm install --save @fortawesome/react-fontawesome
[d3-simple-editor]$ npm install --save @fortawesome/free-brands-svg-icons
[d3-simple-editor]$ npm install --save @fortawesome/free-regular-svg-icons
```

Install.

```
[d3-simple-editor]$ npm install
```

Run the server.
```
[d3-simple-editor]$ node server.js
```

Run the client.
```
[d3-simple-editor]$ npm start
```
